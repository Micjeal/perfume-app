A modern, elegant perfume e-commerce website with a cohesive design system and seamless user experience.

### 01-home (Homepage)

**Purpose:** Interactive quiz to help users discover their perfect scent
**Features:**
- Multi-step questionnaire
- Progress indicator
- Visual option cards
- Personality-based recommendations

### 05-checkout (Secure Checkout)
**Purpose:** Complete purchase with shipping and payment information
**Features:**
- Guest/Login toggle
- Contact and shipping forms
- Multiple payment methods (Card, Apple Pay, PayPal)
- Order summary with expandable details
- Secure payment indicators

### 06-cart (Shopping Cart)
**Purpose:** Review and manage items before checkout
**Features:**
- Quantity adjustments
- Item removal
- Promo code input
- Price breakdown
- Order benefits display

### 07-profile (User Profile)
**Purpose:** Manage account, orders, and preferences
**Features:**
- Profile customization
- Order history
- Favorites management
- Saved addresses
- Notification settings
- Dark mode toggle

### 08-wishlist (My Wishlist)
**Purpose:** Save and manage favorite products for future purchase
**Features:**
- Wishlist statistics (item count, total value)
- Grid view of saved products
- Quick add to cart from wishlist
- Sale badges on discounted items
- Wishlist sharing capability
- Bulk actions (Clear all, Add all to cart)

### 09-compare (Product Comparison)
**Purpose:** Side-by-side comparison of multiple fragrances
**Features:**
- Horizontal scrollable comparison cards
- Detailed product specifications (Rating, Scent Family, Longevity, Sillage)
- Visual rating bars and indicators
- Best for occasion tags
- Quick summary panel (Best Value, Highest Rated, Best Longevity)
- Add/remove products dynamically

### 10-reviews (Customer Reviews)
**Purpose:** Browse and filter customer feedback and testimonials
**Features:**
- Overall rating summary with 5-star breakdown
- Rating distribution visualization
- Filter by all reviews, with photos, or verified purchases
- Individual review cards with user photos
- Photo galleries in reviews
- Helpful vote system
- Verified badge display
- Floating write review button

## üé® Design System

### Color Palette
- **Primary:** `#f9f506` (Signature yellow accent)
- **Background Light:** `#f8f8f5` (Warm off-white)
- **Background Dark:** `#23220f` (Rich dark olive)
- **Surface Light:** `#ffffff` (Pure white)
- **Surface Dark:** `#2c2b1b` (Dark warm gray)

### Typography
- **Font Family:** Spline Sans
- **Weights:** 300 (Light), 400 (Regular), 500 (Medium), 600 (Semi-bold), 700 (Bold)

### Border Radius
- Default: 1rem
- Large: 2rem
- Extra Large: 3rem
- Full: 9999px (Pills/Circles)

### Key Design Principles
1. **Elegant Simplicity:** Clean layouts with ample whitespace
2. **Bold Accents:** Strategic use of signature yellow for CTAs and highlights
3. **Smooth Transitions:** Consistent hover and active states with enhanced animations
4. **Dark Mode Support:** Full theme switching capability across all pages
5. **Mobile-First:** Optimized for touch interactions and small screens
6. **Enhanced Interactivity:** Parallax effects, scale transforms, and smooth scrolling
